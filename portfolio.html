Projeto: Site para Ótica

Sobre: O projeto consiste no desenvolvimento de um sistema web para uma loja óptica, com foco na simulação de um ambiente comercial real e na aplicação prática de conhecimentos em desenvolvimento web. A solução foi implementada utilizando HTML5 para estruturação das páginas, CSS3 para estilização e responsividade, e JavaScript para controle das regras de negócio e interatividade do sistema. O sistema é composto por múltiplos módulos, incluindo catálogo de produtos ópticos, carrinho de compras com cálculo automático do valor total, cadastro de clientes por e-mail e processo de finalização de compras. Cada venda é registrada com data, itens adquiridos e valor total, possibilitando a geração de recibos em formato PDF e a visualização do histórico de compras. O armazenamento de dados é realizado por meio do LocalStorage, simulando a persistência de informações sem a utilização de um banco de dados relacional. A aplicação também dispõe de um painel administrativo protegido por autenticação, permitindo o gerenciamento de clientes cadastrados, acompanhamento do faturamento total e consulta ao histórico completo de vendas. Além disso, o projeto adota boas práticas de organização de código e design responsivo, garantindo compatibilidade com diferentes dispositivos e tamanhos de tela. Este projeto contribuiu para o desenvolvimento de competências técnicas em programação web, lógica de programação, organização de sistemas e compreensão de fluxos comerciais, representando uma solução funcional e didática para demonstração de habilidades em desenvolvimento de software no contexto acadêmico e profissional.

Projeto: Ventilador automático

Sobre: Este projeto consiste no desenvolvimento de um ventilador automático acionado por temperatura, utilizando o microcontrolador ESP32, programado por meio da Arduino IDE. O sistema foi projetado para monitorar continuamente a temperatura do ambiente e acionar automaticamente o ventilador sempre que um valor pré-definido for atingido, garantindo maior eficiência térmica e economia de energia. Objetivo: O principal objetivo do projeto foi implementar uma solução automatizada para controle de ventilação, demonstrando conceitos de Internet das Coisas (IoT), automação, leitura de sensores, controle de atuadores e programação embarcada. Funcionamento: Um sensor de temperatura realiza a leitura contínua do ambiente. O ESP32 processa os dados recebidos do sensor. Quando a temperatura ultrapassa o limite configurado no código, o microcontrolador envia um sinal para acionar o ventilador, por meio de um módulo relé ou transistor. Ao retornar para uma temperatura segura, o sistema desliga automaticamente o ventilador. Tecnologias e Ferramentas Utilizadas: Microcontrolador: ESP32 Ambiente de desenvolvimento: Arduino IDE Linguagem: C/C++ (Arduino) Componentes eletrônicos: Sensor de temperatura Ventilador DC ou AC Módulo relé ou transistor Fonte de alimentação Principais Aprendizados: Programação de microcontroladores com ESP32 Leitura e interpretação de dados de sensores Controle de atuadores de forma automatizada Estruturação de código para sistemas embarcados Integração entre hardware e software Aplicações: O projeto pode ser aplicado em ambientes residenciais, racks de servidores, painéis elétricos e sistemas industriais que necessitam de controle térmico automático, contribuindo para a proteção de equipamentos e redução do consumo energético.
